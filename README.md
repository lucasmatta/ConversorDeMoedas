<h1 align="center"> Conversor de Moedas </h1>

Este é um projeto do curso Alura ONE, em que eu usei a API da ExchangeRate-API para criar um conversor de moedas.
Você pode usar praticamente qualquer moeda existente, com exceção apenas do Wan norte-coreano, pois ele não consta no banco de dados do API.
